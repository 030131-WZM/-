## 🚀 部署方法（选择其中一种）

### 方法1：网页直接上传（最简单）
1. **创建GitHub仓库**
   - 访问：https://github.com/new
   - Repository name：`xiaoming-browser`
   - 选择：Public
   - 点击：Create repository

2. **上传文件**
   - 进入仓库页面
   - 点击：Add file → Upload files
   - 拖拽 `deploy` 文件夹中的所有文件
   - 点击：Commit new files

3. **启用GitHub Pages**
   - 点击：Settings
   - 找到：Pages 选项
   - Source选择：Deploy from a branch
   - Branch选择：main
   - 点击：Save

### 方法2：使用GitHub Desktop
1. 下载：https://desktop.github.com
2. 登录GitHub账户
3. File → Add Local Repository
4. 选择：deploy 文件夹
5. Commit message：Initial commit
6. 点击：Publish repository
7. 在GitHub Desktop中点击：Push origin
8. 启用GitHub Pages（同方法1第3步）
