🚀 方法一：网页直接上传（推荐）

这种方法不需要Git命令，直接通过GitHub网页界面上传文件，适合新手使用。

  1. 准备文件
    打开文件夹：C:\Users\90760\.openclaw\workspaceopenclaw config\deploy
    选择以下文件：
    • index.html
    • mobile.html
    • CNAME
    • README.md
    • .gitignore
  2. 创建GitHub仓库
    访问：https://github.com
    点击右上角 "+" → "New repository"
    Repository name：xiaoming-browser
    选择 Public
    点击 "Create repository"
  3. 上传文件
    进入仓库页面
    点击 "Add file" → "Upload files"
    将准备好的文件拖拽到上传区域
    点击 "Commit new files"
  4. 启用GitHub Pages
    进入仓库页面
    点击 "Settings"
    找到 "Pages" 选项
    Source选择 "Deploy from a branch"
    Branch选择 "main"
    点击 "Save"

⚡ 方法二：使用GitHub Desktop

如果你安装了GitHub Desktop，可以使用图形界面进行操作。

  1. 下载并安装GitHub Desktop：https://desktop.github.com
  2. 登录你的GitHub账户
  3. File → Add Local Repository
  4. 选择 deploy 文件夹
  5. Commit message：Initial commit
  6. 点击 Publish repository
  7. 在GitHub Desktop中点击 Push origin
  8. 启用GitHub Pages（同方法一第4步）
