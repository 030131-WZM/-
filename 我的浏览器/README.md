# 我的浏览器 - 使用说明

## 🌟 功能特色

- **智能搜索**：支持多搜索引擎，搜索结果直接在新页面显示
- **快速访问**：内置常用网站快速访问按钮
- **旅行优化**：专门针对旅行内容的搜索和推荐
- **跨设备兼容**：完美适配桌面和手机端
- **美观界面**：现代化的用户界面设计

## 🚀 使用方法

### 1. 启动服务器
双击 `start.bat` 文件启动服务器

### 2. 访问浏览器
- 本地访问：http://localhost:8888
- 网络访问：http://192.168.172.128:8888

### 3. 搜索功能
- 在搜索框中输入关键词
- 点击搜索按钮或按回车键
- 搜索结果将在新页面中显示
- 点击搜索结果直接打开网页

### 4. 快速搜索
- 点击顶部的搜索引擎按钮（Google、百度、必应等）
- 直接在对应搜索引擎中搜索

### 5. 目地搜索
- 点击下方的目的地卡片
- 自动搜索相关内容

## 📱 设备兼容性

- **桌面端**：完美支持Chrome、Firefox、Safari、Edge等浏览器
- **移动端**：完美支持手机浏览器，响应式设计

## 🔧 技术特性

- **Node.js服务器**：基于Node.js构建，稳定可靠
- **跨域支持**：支持跨域访问
- **文件路由**：自动处理静态文件路由
- **错误处理**：完善的错误处理机制
