# 小铭的专属浏览器 - 快速部署脚本

## 🚀 一键部署脚本

### 使用方法：
1. 双击运行 `GitHub Token部署指南.bat`
2. 按照提示创建GitHub Personal Access Token
3. 使用Token完成推送
4. 启用GitHub Pages

### 部署后访问地址：
- 主浏览器：https://030131-wzm.github.io/-
- 移动端：https://030131-wzm.github.io/-/mobile.html

### 注意事项：
- 确保仓库是Public（公开）类型
- Token需要repo权限
- 部署完成后等待1-5分钟生效

## 📱 安卓端使用：
1. 扫描二维码访问
2. 或直接在浏览器输入地址
3. 支持添加到手机主屏幕

## 🔧 故障排除：
- 如果推送失败，检查Token是否正确
- 确保网络连接正常
- 检查仓库权限设置
