# 小铭的专属浏览器 - 云端部署指南

## 🌟 项目简介
这是一个为小铭专属定制的浏览器界面，支持智能搜索和多搜索引擎，具有美观的用户界面。

## 📱 安卓端访问方案

### 方案一：使用GitHub Pages（推荐，免费）

#### 步骤1：创建GitHub仓库
1. 访问 [GitHub](https://github.com)
2. 登录你的账户
3. 点击右上角的 "+" 号，选择 "New repository"
4. 仓库名称输入：`xiaoming-browser`
5. 选择Public（公开）
6. 点击 "Create repository"

#### 步骤2：部署文件
1. 双击运行 `deploy_browser.bat` 文件
2. 按照提示操作，复制GitHub仓库地址
3. 在命令行中粘贴并执行推送命令

#### 步骤3：启用GitHub Pages
1. 进入你的GitHub仓库
2. 点击 "Settings"
3. 找到 "Pages" 选项
4. 在 "Source" 下选择 "Deploy from a branch"
5. Branch选择 "main"
6. 点击 "Save"

#### 步骤4：访问网站
部署完成后，你的浏览器将通过以下地址访问：
```
https://你的用户名.github.io/xiaoming-browser/
```

### 方案二：使用Netlify（更简单，拖拽即可）

#### 步骤1：准备文件
1. 创建一个文件夹，将浏览器文件复制进去
2. 将文件重命名为 `index.html`

#### 步骤2：部署到Netlify
1. 访问 [Netlify](https://netlify.com)
2. 点击 "Drag and drop your site"
3. 将文件夹拖拽到指定区域
4. Netlify会自动部署并生成一个随机域名

#### 步骤3：自定义域名（可选）
1. 在Netlify设置中可以绑定自定义域名
2. 这样你就可以使用固定的域名访问

### 方案三：使用Vercel（推荐，速度快）

#### 步骤1：安装Vercel CLI
```bash
npm install -g vercel
```

#### 步骤2：部署
```bash
vercel
```
按照提示操作即可

## 📱 安卓端使用方法

### 1. 浏览器访问
- 打开安卓手机浏览器
- 输入上述生成的网址
- 即可使用专属浏览器

### 2. 添加到主屏幕
- 在浏览器中访问网站
- 点击菜单按钮（三个点）
- 选择"添加到主屏幕"或"安装应用"
- 这样就可以像APP一样使用

### 3. 二维码访问
我还会为你生成一个二维码，扫描即可快速访问

## 🔧 自定义设置

### 修改搜索引擎
在 `index.html` 文件中可以修改：
- 快速搜索按钮
- 默认搜索引擎
- 添加新的搜索引擎

### 修改界面样式
- 修改CSS样式文件
- 更改颜色主题
- 调整布局和字体

## 📞 技术支持

如果在部署过程中遇到问题，请检查：
1. GitHub账户是否正常
2. 网络连接是否稳定
3. 文件权限是否正确
