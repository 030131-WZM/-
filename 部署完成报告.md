## 🚀 推荐的部署方法

### 方法1：网页直接上传（最简单）
1. **创建GitHub仓库**
   - 访问：https://github.com/new
   - Repository name：`xiaoming-browser`
   - 选择：Public
   - 点击：Create repository

2. **上传文件**
   - 进入仓库页面
   - 点击：Add file → Upload files
   - 拖拽 `deploy` 文件夹中的所有文件
   - 点击：Commit new files

3. **启用GitHub Pages**
   - 点击：Settings
   - 找到：Pages 选项
   - Source选择：Deploy from a branch
   - Branch选择：main
   - 点击：Save

### 方法2：使用Git命令
```bash
cd deploy
git push -u origin main
```
然后启用GitHub Pages（同方法1第3步）
