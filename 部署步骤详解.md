
### 步骤5：输入认证信息

### 步骤6：启用GitHub Pages
1. 访问：https://github.com/030131-WZM/-
2. 点击 **Settings**
3. 找到 **Pages** 选项（左侧菜单）
4. 在 **Source** 下选择 **"Deploy from a branch"**
5. Branch选择 **"main"**
6. 点击 **Save**

## 🔑 获取GitHub Personal Access Token

### 创建Token步骤：
1. 访问：https://github.com/settings/tokens
2. 点击 **"Generate new token"** → **"Generate new token (classic)"**
3. **Token名称**：`xiaoming-browser-deploy`
4. **过期时间**：选择 `90 days` 或 `No expiration`
5. **勾选权限**：勾选 `repo` (完整仓库访问权限)
6. 点击 **"Generate token"**
7. **复制生成的token**（只显示一次，请妥善保存）

## 🔧 故障排除

### 问题1：推送失败
- 检查网络连接
- 确认Token权限正确
- 重新生成Token

### 问题2：GitHub Pages不显示
- 确保代码已推送成功
- 等待5-10分钟
- 检查仓库是否为Public

### 问题3：认证失败
- 确认用户名正确：`030131-WZM`
- 检查Token是否有效
- 重新生成Token
