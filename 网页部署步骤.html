🚀 小铭的专属浏览器 - 网页部署步骤

使用方法1：网页直接上传（最简单）

1
创建GitHub仓库

首先需要在GitHub上创建一个新的公开仓库。

📸 截图位置：GitHub主页 → New repository页面

📝 详细步骤：

  • 访问：https://github.com/new
  • Repository name：xiaoming-browser
  • 选择：Public（公开仓库）
  • 不要勾选"Add a README file"
  • 点击：Create repository
2
上传文件

将准备好的文件上传到GitHub仓库。

需要上传的文件：

📄
index.html
12,041 bytes
📱
mobile.html
13,913 bytes
🌐
CNAME
26 bytes
📝
README.md
549 bytes
🚫
.gitignore
52 bytes
📸 截图位置：仓库页面 → Add file → Upload files

📝 详细步骤：

  • 进入刚创建的仓库页面
  • 点击：Add file → Upload files
  • 将deploy文件夹中的所有文件拖拽到上传区域
  • 在Commit message框中输入：Initial commit
  • 点击：Commit new files
3
启用GitHub Pages

启用GitHub Pages服务，让网站可以通过互联网访问。

📸 截图位置：仓库Settings → Pages

📝 详细步骤：

  • 进入仓库页面
  • 点击：Settings
  • 在左侧菜单中找到 Pages 选项
  • 在Source部分选择：Deploy from a branch
  • Branch选择：main
  • 文件夹选择：/ (root)
  • 点击：Save

🎉 恭喜！部署完成！

你的专属浏览器已成功部署到GitHub Pages
