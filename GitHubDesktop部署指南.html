📥 第0步：下载GitHub Desktop

这是使用GitHub Desktop部署的第一步，需要先下载并安装官方工具

1
安装并登录GitHub Desktop

下载完成后，安装GitHub Desktop并登录你的GitHub账户。

📸 截图位置：GitHub Desktop登录界面

📝 详细步骤：

  • 运行下载的GitHub Desktop安装程序
  • 按照安装向导完成安装
  • 启动GitHub Desktop
  • 使用你的GitHub账户登录
  • 如果还没有GitHub账户，需要先注册
2
添加本地仓库

将deploy文件夹添加为本地Git仓库。

📸 截图位置：GitHub Desktop → File → Add Local Repository

📝 详细步骤：

  • 在GitHub Desktop中点击 File → Add Local Repository
  • 选择 Choose... 按钮
  • 导航到并选择 deploy文件夹
  • 点击 Add Repository
  • 在Commit message框中输入：Initial commit
3
发布到GitHub

将本地仓库发布到GitHub，创建新的公开仓库。

📸 截图位置：GitHub Desktop → Publish repository

📝 详细步骤：

  • 在GitHub Desktop中点击 Publish repository
  • Repository name：xiaoming-browser
  • 描述：小铭的专属浏览器
  • 选择 Public（公开仓库）
  • 不要勾选 "Initialize with README"
  • 点击 Publish repository
4
推送代码

将本地代码推送到GitHub远程仓库。

📸 截图位置：GitHub Desktop → Push origin

📝 详细步骤：

  • 在GitHub Desktop中确认所有文件都显示在左侧
  • Commit message：Initial commit
  • 点击 Commit to main
  • 点击 Push origin 按钮
  • 等待推送完成
5
启用GitHub Pages

启用GitHub Pages服务，让网站可以通过互联网访问。

📸 截图位置：GitHub仓库 → Settings → Pages

📝 详细步骤：

  • 在浏览器中访问你的GitHub仓库：https://github.com/030131-WZM/-
  • 点击 Settings
  • 在左侧菜单中找到 Pages 选项
  • 在Source部分选择 Deploy from a branch
  • Branch选择 main
  • 文件夹选择 / (root)
  • 点击 Save

🎉 恭喜！部署完成！

你的专属浏览器已成功部署到GitHub Pages
